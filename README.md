# Test SejutaCita Crud + Auth
SejutaCita test, crud + auth with NodeJS Nest Framework

## Installation

```bash
$ npm i
```

## Run

```bash
# dev
$ npm run start:dev

# prod
$ npm run start:prod
```

## Guide
- Open swagger document
  [https://localhost:3001/doc](https://localhost:3001/doc)
